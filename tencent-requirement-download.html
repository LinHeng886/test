🚀 安装指南（3 步）

1

下载并解压技能包

点击上方按钮下载 tencent-requirement.zip，解压后得到 3 个文件。

将它们放到技能目录：

~/.workbuddy/skills/tencent-requirement/
├── SKILL.md
├── _meta.json
└── config.yaml.example
2

配置腾讯文档授权

需要 Node.js (v18+)，然后：

# 安装 mcporter
npm install -g mcporter

# 获取 Token: 访问 👉 https://docs.qq.com/scenario/open-claw.html
# 登录后复制 Token，替换下面的 <你的Token>

# 配置授权
mcporter config add tencent-docs "https://docs.qq.com/openapi/mcp" \
  --header "Authorization=<你的Token>" \
  --transport http --scope home

# 验证（应显示 healthy）
mcporter list
3

配置智能表格连接

将 config.yaml.example 重命名为 config.yaml，填写：

smartsheet:
  file_id: "DSk1Bd1JueFZVeFJ6"
  sheet_id: "t00i2h"
  sheet_title: "智能表1"

✅ 完成！对 AI 说"发布需求"即可使用。

✨ 功能特性

🤖 自然语言发布需求，AI 自动整理表格字段
🔍 发布前智能匹配已有需求（类型互补 + 关键词 + 地域）
📋 浏览、搜索需求表，展示完整匹配详情
👥 多人共用同一张智能表格作为需求广场
🔗 需求类型：求职 / 招聘 / 合作 / 寻求导师 / 其他
🌐 表格链接分享后任何人都能查看
